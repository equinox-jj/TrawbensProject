# Tech Stack
* Kotlin.
* Jetpack Compose.
* MVVM (Model-View-ViewModel).
* Coroutine Flow.
* Koin Injection.
* Retrofit.
* Compose Navigation.
* Coil.
* ROOM Database.
* Lottie Animation.
* Multi Modular.
* Clean Architecture.
* Paging 3.
* Gradle Version Catalog.

# Features
- [x] Save game to favorite.
- [x] Search game by name.
- [x] Pagination to load more data.
